# Feature Branch: Coastal Imagery Integration

## What's New

This branch transforms the RipCat site from clean/modern to authentic Santa Barbara surf culture:

### 🏄‍♂️ Design Philosophy
- **"Ah shucks" feel**: Relaxed, unpretentious surf culture
- **Hollister Ranch vibes**: Earthy, organic, weathered aesthetic  
- **Salty dog**: Authentic working waterfront culture
- **Offshore wind**: That perfect Santa Barbara morning energy

### 🎨 Visual Changes
- **Hand-drawn typography** (Permanent Marker, Caveat)
- **Organic color palette** inspired by actual SB coast
- **Rotated elements** for that casual, imperfect surf culture feel
- **Coastal imagery placeholders** for RipCat scene integration
- **Surf spot specific content** (Rincon, Leadbetter, Surf Beach, Morro Bay)

### 📍 Local Santa Barbara Elements
- **Real surf spot names** and descriptions
- **Authentic surf terminology** ("dawn patrol", "groms", "dialed")
- **Central Coast geographic accuracy** (Pt. Conception to Pt. Sur)
- **Working waterfront culture** (fishing, surfing, sailing)

### 🖼️ Image Integration Plan
Ready for the beautiful coastal RipCat illustrations:
- Hero section: Beach scene with RipCat + surfboard
- Rincon Point card: RipCat at the famous point break
- Harbor card: RipCat on SB pier with phone/charts

### 🎯 Target Audience
